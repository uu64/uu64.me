## 2020年は家でのコーディング時間が増えた1年

2019年までは平均して月30〜40時間は残業していたのだけれど、
今年はコロナの影響もあり職場の方針として残業が原則不可になった。

これはちょうどいいと思って浮いた残業時間を使ってコードを書きはじめたところ、
いい感じに習慣化した一年だった。

### 友人とのサービス開発

2020年度前半は色々とご縁があって、友人のサービス開発のお手伝いをしていた。

ReactやTypeScriptにはここではじめて触れたのだけれど、
メンバーの年齢が近かったこともあって気軽にコミュニケーション取りながら
理解を深めることができて、その結果このブログをNextJSで刷新することもできた。

### Write Code Everyday

Write Code Everydayとは、簡単にいうと毎日コードを書く習慣のこと。
元ネタはJohn Resigという方が提唱したもので以下。

https://johnresig.com/blog/write-code-every-day/

友人が取り組んでいるのを見て自分にもできるかなと思って
10月くらいからはじめて今こんな感じ。

![github](/2021/01/github.png "すこし緑が茂ってきたコミットグラフ。")

[dotfiles](https://github.com/uu64/dotfiles)を整備したり、
ネタが思いつかないときは
[Project Euler](https://projecteuler.net/about)の問題を解いたりしてた。


自分はこの手の毎日何かやるっていう活動が長続きしないほうで、
まだ3ヶ月だけど継続するためにいくつか個人的に気をつけていることがある。
例えばコミットグラフの綺麗さにこだわりすぎないとか。
ここらへんの工夫はそのうちどこかでまとめたいと思っている。

## 2021年にやりたいこと

2021年は引き続きアウトプットの仕方を模索してゆきたい。

2020年は月1回程度このブログでやったことをまとめていたけれど、
まだ自分の中にアウトプットに対する敷居みたいなものがあるので
もう少し気軽に高頻度にちょっとしたことでも文章にするなどしたい。
TwitterとかScrapBoxとかの使い方も工夫できないかなと考えている。

あとOSS活動に興味があって何らかの形で関わりたいと思っている。
最近Go言語の勉強を始めたのでそれでCLIツールを書くとか、
普段自分が使っているOSSのissueを対応してみるとかする予定。
